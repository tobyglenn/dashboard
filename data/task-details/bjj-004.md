# bjj-004: Fix BJJLog Session Creation and Save

**Status:** Review  
**Assignee:** Clarity  
**Epic:** NutriTrack BJJ Buddy

## Description

BJJLogScreen must allow creating a new training session (date, duration, type: gi/nogi, techniques drilled, rolling rounds) and saving it to bjjStore.

## Steps

1. Read `nutrition-app/bjj-buddy/src/screens/BJJLog/BJJLogScreen.tsx`
2. Verify form fields: date, duration, gi/nogi toggle, intensity, notes
3. Verify submit calls `bjjStore.addSession()` with correct payload
4. Verify session appears in store and is visible in BJJProgress
5. Fix any broken form state, navigation, or store calls

## Success Criteria
- Can fill out and submit a BJJ session
- Session saves to store
- Session appears in progress view
