# nt-010: Final NutriTrack Integration Test

**Status:** Review  
**Assignee:** Clarity  
**Epic:** NutriTrack BJJ Buddy

## Description

Full end-to-end test of the NutriTrack app — all screens must work with no runtime errors.

## Steps

1. Run: `cd nutrition-app/nutritrack && npx expo export --platform web 2>&1` — must show 0 errors
2. Run: `npx expo start --web --port 8083` and manually test each screen:
   - Onboarding → enter profile
   - Home → shows today's summary
   - Diary → add food, see entry, delete entry
   - FoodSearch → search returns results
   - BarcodeScanner → shows manual input on web
   - RecipeBuilder → create a recipe
   - Biometrics → log weight
   - Trends → charts render
   - NutritionSummary → shows totals
   - Profile → shows saved data
3. Fix any screen that crashes or shows blank

## Success Criteria
- Zero TypeScript/build errors
- All 10+ screens load and function
- Persistence works across refresh
