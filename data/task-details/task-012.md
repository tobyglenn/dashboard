# Weekly Progress Summary Report

## Summary
Auto-generate a weekly summary every Sunday comparing this week's fitness metrics to prior weeks.

## Details
- Aggregate WHOOP strain, recovery, sleep data by week
- Compare: avg strain, total volume, protein consistency, recovery trends
- Generate HTML report + voice summary
- Deploy to GitHub Pages alongside daily reports

## Acceptance Criteria
- [ ] Weekly aggregation logic working for WHOOP + Cronometer + Speediance
- [ ] HTML report with week-over-week comparison charts
- [ ] Voice summary generated and sent via Telegram
- [ ] Cron job scheduled for Sunday 9 AM
